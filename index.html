회원가입 방법 ▼
1. 우측 상단 로그인/회원가입 클릭
2. 회원가입 탭 선택
3. 아이디 입력 후 중복확인
4. 비밀번호 입력 및 확인
5. 이름, 학원명 입력
6. 휴대폰 번호 인증
7. 회원가입 완료!
과제 기록 방법 ▼
1. 로그인 후 입력 탭 선택
2. 학생 이름 및 반 선택
3. 지각 정도, 학습 태도 선택
4. 지난 과제 이행도 선택
5. 오늘 수업내용 및 과제 입력
6. 저장 버튼 클릭
이미지 저장 방법 ▼
1. 목록 탭에서 저장할 기록 선택
2. 카드 우측 저장 버튼 클릭
3. 날짜별 전체 저장은 전체저장 버튼
4. 저장된 이미지를 카카오톡 등으로 공유

Q. 무료로 이용할 수 있나요?
A. 현재 베타 기간 중 무료로 이용 가능합니다.

Q. 데이터가 삭제될 수 있나요?
A. Firebase 클라우드에 안전하게 저장됩니다.

Q. 여러 선생님이 함께 사용할 수 있나요?
A. 네! 각자 계정으로 로그인하면 독립적으로 사용 가능합니다.
